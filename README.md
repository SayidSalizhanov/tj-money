# TJMoney — Система управления личными финансами

Веб-приложение для учёта доходов и расходов, совместного ведения бюджета в группах, постановки финансовых целей и планирования с помощью напоминаний.

## Технологии

| Слой | Технологии |
|------|-----------|
| Backend | Java 11/16, Jakarta EE (Servlets), JDBC |
| Frontend | JSP, JSTL, HTML5, CSS, JavaScript |
| База данных | PostgreSQL + Flyway (миграции) |
| Сборка | Maven |
| Деплой | Docker + Tomcat 10.1 |
| Внешние сервисы | Cloudinary (аватары), SMTP (email-уведомления) |

## Основной функционал

### Пользователи
- Регистрация и аутентификация (сессии)
- Управление профилем: смена пароля, загрузка аватара через Cloudinary
- Настройка уведомлений (email / Telegram)

### Группы
- Создание финансовых групп и управление ими
- Система заявок на вступление и одобрения участников
- Ролевая модель: администратор / участник
- Режим просмотра группы для внешних пользователей

### Транзакции
- Добавление доходов и расходов (личных и групповых)
- Категории: питание, транспорт, коммунальные услуги, другое
- Фильтрация по периодам: за всё время / месяц / неделю
- Массовый импорт транзакций из Excel-файла (Apache POI)

### Цели
- Создание финансовых целей (личных и групповых)
- Отслеживание прогресса выполнения

### Записи
- Создание заметок/записей внутри группы
- Хранение истории изменений (дата создания и обновления)

### Напоминания
- Создание напоминаний с указанием даты отправки
- Доставка уведомлений на email через SMTP

### Главная страница
- Лента образовательных статей о личных финансах

## Запуск

### Локально

1. Создайте базу данных PostgreSQL:
   ```sql
   CREATE DATABASE "tj-money";
   ```

2. Укажите параметры подключения в `src/main/resources/database.properties`:
   ```properties
   db.url=jdbc:postgresql://localhost:5432/tj-money
   db.username=postgres
   db.password=postgres
   ```

3. Соберите и запустите на Tomcat:
   ```bash
   mvn clean package
   # Разверните target/TJMoney-1.0-SNAPSHOT.war на Tomcat 10.1
   ```

### Через Docker

```bash
docker build -t tjmoney .
docker run -p 8080:8080 tjmoney
```

После запуска приложение доступно по адресу: `http://localhost:8080`

## База данных

Схема создаётся автоматически при первом запуске через Flyway. Основные таблицы:

| Таблица | Назначение |
|---------|-----------|
| `users` | Пользователи |
| `groups` | Финансовые группы |
| `group_members` | Участники групп (с ролями) |
| `transactions` | Транзакции доходов/расходов |
| `goals` | Финансовые цели |
| `records` | Заметки группы |
| `reminders` | Напоминания |
| `applications` | Заявки на вступление в группу |
| `articles` | Образовательные статьи |
| `avatars` | Аватары пользователей |
